# prison-escape-game
A text-based adventure game built in Java where you navigate through a prison and try to escape.
What it does
This is an interactive command-line game where players type commands to move through rooms, collect items, and solve puzzles to escape from a prison. You need to find keys, avoid guards, and navigate the prison layout to win.

Features
Room navigation system - Move between different rooms using directional commands
Inventory management - Pick up and use items like keys and tools
Command parser - Interprets player text input and executes actions
Puzzle mechanics - Solve challenges to progress through the game
Game objects - Various items and obstacles throughout the prison
Scoring system - Tracks your progress and actions
